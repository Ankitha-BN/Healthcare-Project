# Healthcare Analytics-Project (Interactive Dashboard creation using MS Excel/Power BI/Tableau)
This Healthcare Analytics project focuses on analyzing hospital and patient-related data to generate meaningful insights that support better clinical and operational decision-making. The project covers the complete data analytics lifecycle, from data collection and preparation to KPI analysis and interactive dashboard creation using Excel, SQL, Power BI, and Tableau.

**Project Objectives** 
- Analyze healthcare data related to patients, doctors, visits, treatments, diagnoses, and lab results.
- Evaluate treatment effectiveness and patient trends.
- Identify high-risk patients for proactive care.
- Build interactive dashboards highlighting key performance indicators (KPIs).
- Provide insights to improve patient care quality, resource allocation, and hospital efficiency.
